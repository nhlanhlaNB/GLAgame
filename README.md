# GritLab Africa — Launching Soon site

This is a minimal "launching soon" static site for the GritLab Africa game.

How to use
- Place the attached image (the poster you provided) into `assets/cover.jpg`.
- Open `index.html` in your browser to preview locally.

Files created
- `index.html` — main page
- `styles.css` — styling
- `assets/placeholder.txt` — instructions for the image file

If you want, I can embed the image directly into the HTML (base64) so you don't need to add files, or add a simple countdown/subscribe form.
